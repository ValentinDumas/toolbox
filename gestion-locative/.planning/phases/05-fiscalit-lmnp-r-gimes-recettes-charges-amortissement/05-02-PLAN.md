<objective>
Livrer la première slice utilisateur Phase 5 : qualifier les Justificatifs et
Tickets en 4 catégories alignées 2033-A (entretien_reparation / amelioration /
charge_courante_periodique / non_deductible), agréger recettes et charges
annuelles, calculer l'abattement micro-BIC (50 % longue durée, plancher 305 €,
seuil 83 600 €). L'utilisateur peut maintenant qualifier ses charges et voir le
résultat micro-BIC pour une année donnée.

Purpose : FIS-02 (abattement micro-BIC) + FIS-03 partiel (agrégation recettes
& charges). C'est le slice utilisateur le plus simple : pas d'amortissement,
pas de clôture, pas de LMP — juste la qualification + le calcul micro. Permet
de valider la chaîne UI → use case → repos étendus → domaine en isolation,
avant d'empiler la complexité aux Plans 03+.

Output :
  - Justificatif + TicketTravaux étendus avec qualification (4 catégories)
  - 3 use cases : qualifier-ticket, decomposer-justificatif, calculer-micro-bic
  - Repos sommeRecettesAnnuelles + sommeChargesParCategorie (read-only)
  - Page S5 fonctionnelle + 3 widgets pédagogiques (TF/TEOM, syndic, multi-biens) + badge 'Sans PJ'
  - Couverture BDD @fis-02 (4 scénarios CONTEXT) + @fis-03 (3 scénarios qualif/split + 1 badge)
</objective>
