Statut: 'en_attente' | 'partiellement_payee' | 'payee' | 'annulee' (D-55). Invariant : total = loyerHc + montantCharges (snapshot D-54). Méthode `avecStatut(statut)` copy-on-write.

- `PdfRendererPdfmake` adapter : `createRequire(import.meta.url)` (RESEARCH §Topic 1) + Roboto TTF embedded (accents FR) + `setUrlAccessPolicy(() => false)` + `setLocalAccessPolicy(() => true)`.

Algorithme prorata 1ère échéance (RESEARCH §Topic 4 calculerProrataPremiereEcheance) :
- `joursInMois = BigInt(dateDebut.daysInMonth)`
- `jourDebutOccupation = BigInt(dateDebut.day)`
- `joursOccupes = joursInMois - jourDebutOccupation + 1n`
- `loyerProrata = loyerHc.multiplyByFraction(joursOccupes, joursInMois)`
- Idem charges
- periodeDebut = dateDebut, periodeFin = dateDebut.with({ day: dateDebut.daysInMonth })
- `jourEcheanceAttendue = dateDebut.with({ day: jourEcheance })` (sauf si dateDebut.day > jourEcheance → mois suivant)

Algorithme dernière échéance prorata si fin de bail tombe en milieu de mois : analogue mais `joursOccupes = jourFin.day` (du 1er du mois à jourFin).
